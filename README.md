# AI-tools Moonshot Project
This repository contains the materials for my moonshot project for the AI Tools course.

The project explores whether an AI tool (ChatGPT, GPT-5.3) can generate a simple data science workflow for stock market forecasting using Apple stock data.

## Contents

- **ai_stock_workflow_R.R**  
  R code generated with the help of ChatGPT to build a stock forecasting workflow (including both initial and revised version).

- **aapl_workflow_comparison.png**  
  Visualization comparing the initial AI-generated workflow and the revised workflow.

- **ai_stock_results_summary.txt**  
  Summary of model outputs and accuracy comparison.

- **ai_moonshot_reflection.pdf**  
  Written reflection discussing what worked, what didn’t, and how the AI-generated output was evaluated.

## AI Tool Used
ChatGPT (OpenAI GPT-5.3)
